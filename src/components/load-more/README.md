# API

```jsx
import { KLoadMore } from "kui-weapp";
```

## KLoadMore

| 名称      | 类型            | 默认值 | 描述           |
| --------- | --------------- | ------ | -------------- |
| tip       | string          | -      | 提示           |
| renderTip | React.ReactNode | -      | 提示           |
| loading   | boolean         | true   | 是否加载中     |
| full      | boolean         | -      | 宽度是否百分百 |
