# API

```jsx
import { KIcon } from "kui-weapp";
```

## KIcon

| 名称      | 类型     | 默认值 | 描述                                                           |
| --------- | -------- | ------ | -------------------------------------------------------------- |
| fontSize  | number   | -      | 图标大小                                                       |
| color     | string   | -      | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
| prefixCls | string   | k-icon | 图标前缀                                                       |
| type      | string   | -      | 图标类型                                                       |
| onClick   | ()=>void | -      | 点击回调                                                       |
