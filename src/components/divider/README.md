# API

```jsx
import { KDivider } from "kui-weapp";
```

## Divider

| 名称     | 类型    | 默认值 | 描述                                                           |
| -------- | ------- | ------ | -------------------------------------------------------------- |
| vertical | boolean | -      | 是否垂直                                                       |
| color    | string  | -      | 颜色风格，可选值 'primary' 'info' 'success' 'warning' 'danger' |
